# Neomem

An information manager


## Goals

- 
- open source, with free/paid hosting plans
- make a plugin ecosystem for data sources and ui elements - free/paid


## Features

- doc, grid/table, and tree/outline views of same data
- graphql backend with plugins for different data sources
- cloud db as another datasource - could host certain objects online to publish or share with others [paid]


## Use Cases

- task manager - projects, tasks, goals
- art travel planner - location (continent/country/state/city/museum), artist, date, name, rating, source, size, images
- screenplay/outline editor - acts, scenes
