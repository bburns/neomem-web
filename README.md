# Neomem

A document/database information manager


## Goals

- document, grid/table, and tree/outline views of same information
- all views are plugins - others can be created, e.g. map view, calendar view, kanban view
- open source, with free/paid hosting plans
- plugin ecosystem for data sources and ui elements - free/paid


## Use Cases

- task manager - projects, tasks, goals, timeframe, order, estimate, actual
- art travel planner - location (continent/country/state/city/museum), artist, date, name, rating, source, size, images
- screenplay/outline editor - acts, scenes


## Implementation

- graphql backend with plugins for different data sources
- cloud db as another datasource - host certain objects online to publish or share with others
